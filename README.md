# Ansible Docker Role

An Ansible Role that installs [Docker](https://www.docker.com) on Linux.

## Role Variables

Available variables with their default values are defined in [defaults/main.yml](defaults/main.yml).

## Supported Platform

* Centos 7
* Ubuntu 16.04
